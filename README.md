# 📊 Projeto de Análise de Dados

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar análise e manipulação de dados utilizando Java com Spring Boot.

A aplicação consome uma API pública do IBGE para obter informações sobre os estados brasileiros e seus respectivos municípios. A partir desses dados, é feito o processamento, organização e persistência das informações em banco de dados.

---

## 🚀 Tecnologias utilizadas

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* Maven

---

## 🔗 API utilizada

* IBGE (API pública)
  A API foi utilizada para consultar dados como:
* Quantidade de municípios por estado
* Consulta por sigla do estado (UF)
* Listagem de municípios

---

## 📂 Estrutura do projeto

* `/controller` → responsáveis pelas requisições da aplicação
* `/service` → regras de negócio e consumo da API
* `/repository` → comunicação com o banco de dados
* `/model` → entidades do sistema
* `/config` → configurações gerais

---

## 🔍 Funcionalidades

* Consumo de API pública (IBGE)
* Processamento de dados de estados e municípios
* Armazenamento em banco de dados
* Consulta de informações por estado
* Organização em arquitetura em camadas

---

## 📈 Objetivo

O foco deste projeto é consolidar conceitos de desenvolvimento backend com Java, especialmente no consumo de APIs externas, tratamento de dados e persistência em banco.

---



## 📌 Observações

Os dados utilizados são provenientes de uma API pública do IBGE, podendo sofrer alterações conforme atualizações da própria fonte.

Este projeto faz parte dos meus estudos em desenvolvimento backend e análise de dados.
